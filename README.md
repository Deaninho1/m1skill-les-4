## Werken met een online repo

* maak git repository (voeg readme file toe)
* kopieër de link in je repository
* open je terminal op je computer
* in je terminal ga naar de directory waar je de repository wil opslaan (maak een nieuwe als het moet)
* kopieër je repo naar je pc met git clone en de repository link
* als je klaar bent met nieuwe dingen toevoegen typ git status in je terminal om te checken of de je nieuwe changes hebt
* als je rode text ziet typ git add om de changes te stagen
* typ daarna git commit -m *naam van de commit* om je changes to commiten
* push daarna met git push -u origin main (latere commits kan je pushen met git push)
